# 🔐 Cipher System

A web-based cryptography tool built with **Vite**, **React**, and **TypeScript**. It allows users to encrypt and decrypt text using several classical cipher algorithms through a simple and responsive interface.

## 🚀 Features

- Supports multiple ciphers:
  - Pigpen
  - Affine
  - Playfair
  - Hill
  - Vigenère  
- Real-time encryption and decryption  
- Clean and user-friendly UI  
- Fast performance with Vite  

## 🛠️ Tech Stack

- React  
- TypeScript  
- Vite  

## 📦 Installation

```bash
git clone https://github.com/cyfuu/Cipher-System.git
cd Cipher-System
npm install
npm run dev
```

## 🧪 Usage

1. Select a cipher  
2. Input your message  
3. Enter the required key (if applicable)  
4. View the encrypted or decrypted output  

## 📚 Supported Ciphers

- **Pigpen** – Symbol-based substitution cipher  
- **Affine** – Uses a mathematical function for substitution  
- **Playfair** – Encrypts letter pairs using a 5×5 grid  
- **Hill** – Matrix-based encryption method  
- **Vigenère** – Uses a repeating keyword for shifting letters
